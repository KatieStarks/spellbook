## Usage

1. Run the Android Figwheel Configuration. (play button intelliJ)

2. Open terminal here:

```
emulator -avd Nexus_5X_API_24_x86
```

3. In new terminal window:

```
react-native run-android
```

Restart on crash:
repeat 2. and 3.
